== Description ==

Bindet Geo-Tags, in den &lt;head&gt;-Bereich ein.

== Installation ==

1. 'blogmappr.php' nach '/wp-content/plugins/' hochladen
1. Unter 'Plugins' im Wordpress Admin-Interface das Plugin aktivieren
1. Unter 'Einstellungen->Blogmappr' die Daten des Blogs eingeben und speichern.
